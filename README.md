# 🎮 Game Registration

Rastreador pessoal de jogos com contas individuais na nuvem.
Desenvolvido por **Antonio Carvalho — ICMC/USP**.

---

## 📖 Historia

Este app nasceu de uma necessidade pessoal: antes eu controlava meus jogos em planilhas. Aproveitei para colocar em prática o que estava aprendendo e o projeto acabou crescendo além do esperado — de uma ferramenta só minha para um app público.

## ✨ Funcionalidades

- **Login e cadastro** — cada usuário tem sua própria conta e dados isolados na nuvem
- **Dashboard** — total de jogos, horas jogadas, jogos zerados e gênero favorito
- **Adicionar Jogo** — nome, plataforma, gênero, status, nota, review, datas e horas
- **Busca automática de capas** — Steam, Nintendo, PlayStation e Xbox
- **Lista de Jogos** — filtros, edição e exclusão inline
- **Coleções/Pastas** — organize seus jogos em pastas com ícone e cor
- **Estatísticas** — gráficos e rankings detalhados
- **Modo escuro/claro**
